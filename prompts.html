База полезных промтов для LLM

Здесь собраны промты, которые я использую в работе с ChatGPT, DeepSeek и другими большими языковыми моделями. Вы можете взять их за основу, адаптировать под свои задачи или просто изучить примеры эффективных запросов.

Разработка

Промт для создания базовой структуры HTML страницы

Этот промт помогает быстро сгенерировать шаблон HTML5 страницы с мета-тегами, подключением CSS и JS.


Создай базовый шаблон HTML5 страницы.
Включи:
- Doctype HTML5
- Тег html с lang="ru"
- Тег head с charset="UTF-8" и viewport meta tag.
- Тег title (поставь "Пример страницы")
- Тег meta description (с описанием "Это пример сгенерированной страницы")
- Подключение внешнего CSS файла (style.css)
- Подключение внешнего JS файла (script.js) в конце body.
- Внутри body добавь пустой div с id="app".
Промт для написания простого PHP скрипта

Генерирует очень простой PHP скрипт для быстрого теста.


Напиши простой PHP скрипт, который выводит на экран строку "Привет, мир!" и текущую дату в формате ГГГГ-ММ-ДД.

Контент

Промт для генерации идеи статьи для блога

Используйте этот промт для брейнсторминга тем, подходящих для вашей аудитории.


Я пишу блог о [Ваша тема блога, например: "эффективное использование LLM в работе"].
Моя целевая аудитория - [Описание ЦА, например: "фрилансеры и малый бизнес, которые хотят автоматизировать рутинные задачи"].
Предложи 5 оригинальных идей для статей или постов в блог, которые будут максимально полезны этой аудитории.
Для каждой идеи укажи:
1. Заголовок
2. Краткое описание (2-3 предложения)
3. Примерное ключевое слово или фраза для SEO.
Промт для написания продающего текста для продукта (AIDA)

Шаблон для создания убедительного текста по классической маркетинговой формуле AIDA.


Напиши продающий текст для моего [Продукт/Услуга: например, "онлайн-курс по быстрому освоению промт-инжиниринга"].
Целевая аудитория: [Описание ЦА: например, "новички в ИИ, которые хотят быстро получить практические навыки"].
Основные преимущества продукта/услуги:
- [Преимущество 1: например, "Минимум теории, максимум практики"]
- [Преимущество 2: например, "Создание реальных проектов уже на курсе"]
- [Преимущество 3: например, "Поддержка автора"]
Призыв к действию (CTA): [Что должен сделать пользователь: например, "Записаться на курс", "Скачать пробный урок", "Оставить заявку"].
Длина текста: [Например, "около 300 слов", "не более 5 абзацев"].
Используй структуру AIDA (Attention, Interest, Desire, Action) для построения текста. Сделай тон текста [Тон: например, "дружелюбным и мотивирующим", "профессиональным и убедительным"].

Писать как человек

Промт для переписывания текста в более разговорном стиле

Полезно, когда нужно сделать официальный или технический текст более живым и доступным.


Перепиши следующий текст в более неформальном, разговорном стиле, как если бы ты объяснял это другу или рассказывал историю в своем блоге. Используй простые слова, избегай канцеляризмов и сложных оборотов. Добавь немного эмоций или личного отношения, если это уместно.

[Вставьте текст, который нужно переписать, сюда]
Промт для имитации определенной личности или стиля письма

Позволяет настроить LLM на определенный тон, словарный запас или даже манеру речи.


С этого момента и во всех следующих ответах имитируй манеру речи и стиль письма [Опишите личность или стиль: например, "опытного, но немного саркастичного технического специалиста", "дружелюбного и восторженного новичка", "официального представителя компании"].
Твой словарный запас должен соответствовать этой роли. Избегай стандартных фраз ИИ.
Начни с ответа на вопрос: [Задайте первый вопрос, чтобы начать диалог в нужном стиле].
